# Dark Photon (vector portal) — Active Workflow

Dark photon A': massive spin-1 vector boson with kinetic mixing epsilon with SM photon.

Two production channels:
- **Heavy A'**: exotic Higgs decay `pp → h(125) → A'A'` — mass benchmarks:
  - `generator/heavy_dp_m05.cmnd` — m_A' = 0.5 GeV
  - `generator/heavy_dp_m1.cmnd`  — m_A' = 1 GeV
  - `generator/heavy_dp_m15.cmnd` — m_A' = 15 GeV
- **Light A'** (sub-GeV): meson decay `pp → X → η/ω → A' + γ/π⁰` (kinetic mixing)
  - `generator/meson_dp_eta_m05.cmnd` — η→A'γ, m_A' = 0.5 GeV
  - `generator/meson_dp_omega_m05.cmnd` — ω→A'π⁰, m_A' = 0.5 GeV
  - Production via `SoftQCD:nonDiffractive`, σ = σ_inel ≈ 80 mb
- **Light A'** (1 GeV): Drell-Yan `pp → qq̄ → γ* → A'` (kinetic mixing)
  - `generator/dy_dp_m1.cmnd` — m_A' = 1.0 GeV (all meson channels kinematically closed)
  - Production via `NewGaugeBoson:ffbar2gmZZprime` (PDG 32), pure Z' mode
  - σ at ε²=1 from Pythia8 × K≈8.09; analysis multiplies by ε² in `--production drell_yan` scan

Decay: kinetic-mixing BRs from the R-ratio (mass-dependent leptonic + hadronic channels).
Pythia8 handles full decay and hadronization. Generator writes charged daughters to `_daughters.csv`.

## Branching Ratios

Cmnd files are **generated by `generator/make_dp_cmnd.py`** — do not edit them by hand.

| Mass range | Method | Source |
|---|---|---|
| 0.2–1.7 GeV | Vector Meson Dominance (VMD) | DeLiVeR ([arXiv:2201.01788](https://arxiv.org/abs/2201.01788)) |
| > 1.7 GeV | Perturbative QCD, NLO αs | R-ratio formula with 1-loop αs running |

The pre-computed VMD table is `generator/br_tables/dp_brs_deliver.csv` (25 mass points, 0.20–1.70 GeV).
Channels below 0.5% of total are merged into π⁺π⁻; BRs are renormalised to sum to 1.

```fish
# Regenerate a heavy DP cmnd file at any mass:
conda run -n llpatcolliders python generator/make_dp_cmnd.py \
    --mass 0.5 --tau0 1e4 --outfile generator/heavy_dp_m05.cmnd

# Generate meson DP cmnd files (light A' via meson decay):
conda run -n llpatcolliders python generator/make_dp_cmnd.py \
    --production meson --mass 0.5 --parent omega --outfile generator/meson_dp_omega_m05.cmnd
conda run -n llpatcolliders python generator/make_dp_cmnd.py \
    --production meson --mass 0.5 --parent eta --outfile generator/meson_dp_eta_m05.cmnd

# Generate Drell-Yan DP cmnd files (light A' at masses where mesons are closed):
conda run -n llpatcolliders python generator/make_dp_cmnd.py \
    --production drell_yan --mass 1.0 --outfile generator/dy_dp_m1.cmnd

# Regenerate the VMD BR table from a local DeLiVeR clone:
conda run -n llpatcolliders python generator/generate_br_table.py \
    --deliver-path /path/to/DeLiVeR --outfile generator/br_tables/dp_brs_deliver.csv

# Validate the pipeline (table vs DeLiVeR direct):
conda run -n llpatcolliders python generator/validate_brs.py \
    --deliver-path /path/to/DeLiVeR
```

Validation result (Phase 2): 47/49 channel comparisons pass the acceptance criteria
(10% outside resonances, 20% inside ρ/ω/φ windows). All three cmnd file masses (0.5, 1.0, 15 GeV)
pass at < 1%. See `output/data/dp_br_validation.csv` and `output/images/dp_br_vs_mass.png`.

## Production

```fish
set DP /Users/fredi/sandbox-offline/llpatcolliders_MATT/llpatcolliders/dark_photon

# h→A'A', three mass points (σ(gg→h) ≈ 54.7 pb N3LO 14 TeV = 54700 fb)
# Argument 2 is target generated events (not LLP rows).
# Argument 4 is max parallel generator jobs (roughly max CPU cores used).
bash $DP/generator/produce.sh $DP/generator/heavy_dp_m05.cmnd 50000 dp_heavy_m05 4
bash $DP/generator/produce.sh $DP/generator/heavy_dp_m1.cmnd  50000 dp_heavy_m1  4
bash $DP/generator/produce.sh $DP/generator/heavy_dp_m15.cmnd 50000 dp_heavy_m15 4

# Light A' via meson decay (σ_inel ≈ 80 mb, SoftQCD)
# Separate samples per parent meson for clean weighting:
bash $DP/generator/produce.sh $DP/generator/meson_dp_omega_m05.cmnd 100000 dp_meson_omega_m05 4
bash $DP/generator/produce.sh $DP/generator/meson_dp_eta_m05.cmnd   100000 dp_meson_eta_m05   4

# Light A' via Drell-Yan (m=1.0 GeV, σ at ε²=1 from Pythia8)
bash $DP/generator/produce.sh $DP/generator/dy_dp_m1.cmnd 10000 dp_dy_m1 4

# Output structure:
#   data   -> LLP CSV, daughters CSV, meta JSON
#   images -> plots from analysis scripts
```

## Analysis (conda env llpatcolliders)

Always pass `--xsec` explicitly.

```fish
set DP /Users/fredi/sandbox-offline/llpatcolliders_MATT/llpatcolliders/dark_photon
cd $DP

# HL-LHC baseline: sqrt(s)=14 TeV, L=3000 fb^-1
for tag in dp_heavy_m05 dp_heavy_m1 dp_heavy_m15
    # N-track analysis with pair+separation selection:
    #   >=2 charged daughters with p>600 MeV, union of all valid pair separation windows [1 mm, 1 m]
    conda run -n llpatcolliders python decayProbPerEvent_Ntrack.py \
        output/data/$tag.csv --xsec 54700 --lumi 3000 --outdir output/ \
        --sep-min 0.001 --sep-max 1.0
    conda run -n llpatcolliders python signal_surface_hitmap_v2.py \
        output/data/$tag.csv --outdir output/
end

# Meson production: ε² scan (σ_inel ≈ 80 mb = 8e10 fb)
for parent in omega eta
    conda run -n llpatcolliders python decayProbPerEvent_Ntrack.py \
        output/data/dp_meson_{$parent}_m05.csv \
        --xsec 80000000000 --lumi 3000 --outdir output/ \
        --sep-min 0.001 --sep-max 1.0 \
        --production meson --parent-meson $parent --dp-mass 0.5
    conda run -n llpatcolliders python signal_surface_hitmap_v2.py \
        output/data/dp_meson_{$parent}_m05.csv --outdir output/
end

# Drell-Yan production: ε² scan (σ at ε²=1 from Pythia8 log, convert mb→fb)
conda run -n llpatcolliders python decayProbPerEvent_Ntrack.py \
    output/data/dp_dy_m1.csv \
    --xsec <sigma_fb_from_log> --lumi 3000 --outdir output/ \
    --production drell_yan --dp-mass 1.0

# Geometry validation (after any change to gargoyle_geometry.py):
conda run -n llpatcolliders python visualize_tunnel.py
```

## Key differences from ALP

| Property        | ALP (`6000113`)  | Dark photon (`6000115`) |
|-----------------|------------------|-------------------------|
| Spin            | 0 (pseudoscalar) | 1 (vector)              |
| Pythia spinType | 1                | 3                       |
| Particle name   | ALP              | Aprime                  |
| isResonance     | off              | off                     |
| Production      | same (h→XX)      | same                    |
| Decay           | μ⁺μ⁻ (BR=1)      | R-ratio BRs (full)      |
| Analysis        | 2-body analytical | >=2 charged daughters + union of all valid pair separation windows [1 mm, 1 m] |

## External comparison curves

`external/` accepts digitised sensitivity curves from ANUBIS/CODEX-b/MATHUSLA dark photon papers.
The existing h→SS dark-Higgs curves (from the MATHUSLA/ANUBIS/CODEX-b proposals) are a valid
geometric proxy for h→A'A' at the same mass — same production mechanism, same boost distribution.
No dedicated h→A'A' curves at m=15 GeV have been published yet.
